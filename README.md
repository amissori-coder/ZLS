# ZLS - Zone Logistiche Semplificate | Revilaw SPA

Sito web informativo sulle agevolazioni fiscali delle Zone Logistiche Semplificate (ZLS), realizzato per **Revilaw SPA** - Società di Revisione Legale.

## Struttura

- `index.html` - Pagina principale del sito
- `style.css` - Stili CSS (tema scuro professionale)
- `script.js` - Interattività (menu mobile, animazioni scroll, form contatti)

## Sezioni

1. **Hero** - Presentazione dell'agevolazione ZLS con CTA
2. **Cosa sono le ZLS** - Spiegazione dell'agevolazione fiscale
3. **Vantaggi** - Benefici concreti per le imprese
4. **Chi ne beneficia** - Destinatari (piccole, medie e grandi imprese)
5. **Perché Revilaw** - Servizi offerti dalla società
6. **Contatti** - Form di contatto e informazioni

## Come usare

Aprire `index.html` in un browser per visualizzare il sito.
